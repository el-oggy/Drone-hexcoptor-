# 🚁 Custom Hexacopter Flight Controller

A custom flight controller developed as part of my embedded systems and PCB design learning journey. This project focuses on designing a reliable flight controller for a hexacopter using an STM32 microcontroller, integrating multiple sensors and communication interfaces commonly used in modern UAV systems.

> **Note:** This repository is currently under active development. Documentation and design files will be added as the project progresses.

---

## 📖 Overview

The goal of this project is to design and develop a custom flight controller capable of supporting stable and reliable hexacopter operation. The project combines embedded systems, PCB design, sensor integration, and firmware development into a single hardware platform.

This repository serves as a central place to document the development process, hardware design, firmware, and future improvements.
